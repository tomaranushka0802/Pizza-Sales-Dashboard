# Pizza Sales Dashboard | Power BI

An interactive Pizza Sales Analysis Dashboard built using SQL Server, Power BI, and DAX.

## Project Overview
This project analyzes pizza sales data to understand revenue, order trends, best-selling pizzas, category performance, and customer ordering patterns.

## Key Insights
- Analyzed total revenue, total orders, and average order value.
- Identified top-selling pizzas and pizza categories.
- Tracked daily and monthly sales trends.
- Analyzed sales by pizza size and category.
- Created KPI measures using DAX.

## Tools Used
- SQL Server
- Power BI
- DAX
- Excel / CSV Dataset
